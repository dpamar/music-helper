# music-helper

Application web client-side pour saisir et afficher des partitions musicales en notation française.

## Fonctionnalités

- Saisie intuitive en notation française (Do, Ré, Mi...)
- Rendu graphique automatique sur portée musicale (Canvas HTML5)
- Support des altérations (dièse, bémol, bécarre)
- Gestion des durées (ronde, blanche, noire, croche, double-croche, notes pointées)
- Accords (plusieurs notes simultanées)
- Clef de sol et clef de fa
- **Export PNG** : Téléchargez votre partition en image haute qualité
- Interface responsive (desktop, tablet, mobile)

## Utilisation

1. Ouvrez `index.html` dans votre navigateur
2. Saisissez votre partition dans le champ de texte (ou cliquez sur "Charger un exemple")
3. Cliquez sur "Générer la partition"
4. Cliquez sur "Exporter en PNG" pour télécharger votre partition en image

## Déploiement

Aucun serveur requis. Ouvrez simplement `index.html` dans un navigateur moderne (Chrome, Firefox, Safari, Edge).

Pour GitHub Pages : Settings > Pages > Source: main branch.
